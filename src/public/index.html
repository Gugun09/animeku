Anime API Documentation

Welcome to the Anime API! Below are the available endpoints.

GET /api/home

Description: Get a list of all anime.

GET /api/batch

Description: Get batch anime with pagination.

GET /api/terbaru

Description: Get a list of the latest anime.

GET /api/anime/:animeId

Description: Get details of an anime by its animeId. Replace {animeId} with the actual anime ID.
